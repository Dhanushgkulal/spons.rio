# SponsorBridge 🤝

SponsorBridge is an open two-way sponsorship marketplace that eliminates the need 
for offline back-and-forth between event organizers and potential sponsors.

Whether you're organizing a college fest, sports tournament, NGO drive, or corporate 
summit — or a brand looking to grow visibility and community presence — SponsorBridge 
gives both sides a dedicated space to discover, negotiate, and confirm sponsorship 
deals entirely online.

## The Problem It Solves 

Sponsorship in India (and globally) is still heavily dependent on cold calls, 
WhatsApp forwards, and in-person meetings with no structured follow-up. Organizers 
struggle to find the right sponsors; sponsors have no easy way to discover events 
aligned with their goals. SponsorBridge fixes this.

## Who It's For

- **Event Organizers** — Post your event, define sponsorship tiers, receive and 
  manage proposals from interested sponsors, and communicate directly through the platform.

- **Sponsors** — Browse verified events by category, location, and budget. Send 
  proposals, negotiate terms, and track every deal from application to completion.

## Key Features

- Dual-role onboarding (Organizer / Sponsor)
- Event listing with customizable sponsorship tiers
- Sponsor discovery feed with filters
- In-app messaging and deal negotiation room
- Deal confirmation with locked agreement summary
- Post-event reviews and ratings for both parties
- Notification system for every stage of the deal lifecycle

## Tech Stack

- **Frontend:** Next.js / React, Tailwind CSS
- **UI Components:** shadcn/ui, lucide-react
- **Charts:** Recharts
- **Auth:** NextAuth.js
- **Database:** PostgreSQL with Prisma ORM
- **Storage:** Cloudinary (for event posters and brochures)
- **Realtime Messaging:** Socket.io

## Status

🚧 Currently in development. UI built with v0.dev. Backend integration in progress.
